- single - player vs computer
- multiplayer - one keybord - player vs player

## Key bindings
- escape - exit games
- f1 - motion speed up
- f2 - motion speed down
- f5 - reset game 
- f6, p - pause game
- f9 - switch play pc/player mode for left bat
- f10 - switch play pc/player mode for right bat
- f12 - show FPS
- m - mute sounds
